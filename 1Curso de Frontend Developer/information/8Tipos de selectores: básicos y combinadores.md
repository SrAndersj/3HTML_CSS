# SELECTORES BÁSICOS

| SELECTORES BÁSICOS | EJEMPLOS                 |
| ------------------ | ------------------------ |
| DE TIPO            | `div {...}`              |
| DE CLASE           | `.elemento {...}`        |
| DE ID              | `#id-del-elemento {...}` |
| DE ATRIBUTO        | `a[href="..."] {...}`    |
| UNIVERSAL          | `* {...}`                |

# SELECTORES COMBINADORES

| SELECTORES COMBINADORES | EJEMPLOS  |
| ----------------------- | --------- |
| DESCENDIENTES           | `div p`   |
| HIJO DIRECTO            | `div > p` |
| ELEMENTO ADYACENTE      | `div + p` |
| GENERAL DE HERMANOS     | `div ~ p` |
